# Comandos Git

Atualizando projeto máquina local
```
  git pull
```

Verificando alterações
```
  git status
```

Adicionando alterações ao repo local
```
  git add .
```

Comitando as alterações
```
  git commit -m "messagem"
```

Empurrando para o repo remoto
```
  git push
```
